# Apresentação Felipe - Project Context

## Visão Geral
Apresentação comercial interativa da **Produza** para o cliente **Felipe** — empresário de pizzarias, líder no iFood no Nordeste, em expansão nacional para São Paulo.

O objetivo é vender um projeto estratégico de **marketing pessoal e lançamento como influenciador digital** com 3 opções de plano.

## Estrutura (10 slides)
1. **Capa** — Hero com identidade, stats do Felipe
2. **Sobre Felipe** — Timeline de vida e valores
3. **Oportunidade** — Pesquisa de mercado (dados reais)
4. **Diagnóstico** — Onde está vs. onde pode chegar (barras animadas)
5. **Metodologia** — Como a Produza trabalha (4 etapas)
6. **Ecossistema** — 6 pilares interativos (hover para detalhes)
7. **Plano Presença** — R$ 5.980 / One Page
8. **Plano Autoridade** — R$ 12.980 / One Page (DESTAQUE)
9. **Plano Influenciador Pro** — R$ 24.900 / One Page
10. **Contratar** — Tabela comparativa + CTA WhatsApp
